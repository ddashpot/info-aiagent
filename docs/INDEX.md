# 知識ベース体系インデックス

AIエージェントの **開発** と **リスク管理** の知識を、既存データから漏れなく一覧化した体系ドキュメント群。
全ドキュメントは `docs/_generate.py` で再生成可能（要約・省略・改変なし）。

## ドキュメント

| ドキュメント | 内容 |
|---|---|
| [開発体系.md](開発体系.md) | 設計パターン・メモリ・ツール・推奨構成・相性・実装基盤比較 |
| [リスク管理体系.md](リスク管理体系.md) | 脆弱性カタログ・分類軸・フレームワーク・方法論・スキャナ・ID要件 |
